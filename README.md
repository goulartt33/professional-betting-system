# Professional Betting System

Sistema profissional de análise de apostas esportivas com IA integrada.

## 🚀 Funcionalidades

- ✅ Coleta de dados em tempo real de APIs profissionais
- ✅ Análise preditiva com machine learning
- ✅ Geração inteligente de bilhetes
- ✅ Integração com Telegram
- ✅ Deploy automático no Render

## 📋 Pré-requisitos

- Python 3.11+
- Conta no [Football Data API](https://www.football-data.org/)
- Bot do Telegram

## ⚙️ Configuração

1. Clone o repositório
2. Configure as variáveis de ambiente (`.env`)
3. Instale as dependências: `pip install -r requirements.txt`
4. Execute: `python src/main.py`

## 🚀 Deploy no Render

1. Faça push para o GitHub
2. Conecte no Render
3. Configure as environment variables
4. Deploy automático
